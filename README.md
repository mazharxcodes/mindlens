# MindLens 🧠

AI-powered Chrome extension that detects content bias in Instagram feeds and injects balanced perspectives in real-time.

## 🚀 Features
- Real-time content analysis
- Bias detection using AI
- Perspective injection system
- Lightweight Chrome extension

## 🧠 Tech Stack
- TypeScript
- Chrome Extension (Manifest V3)
- OpenAI APIs (Embeddings + LLM)
