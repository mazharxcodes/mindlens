# Contributing to MindLens

Thanks for contributing to MindLens.

## Workflow

1. Fork the repository
2. Create a branch from `main`
3. Make focused changes
4. Run checks locally
5. Open a pull request against `main`

## Local Checks

```bash
npm run typecheck
npm run build
```

## Guidelines

- Prefer small, reviewable pull requests
- Preserve the local-first architecture
- Avoid brittle Instagram-specific selectors when possible
- Update docs when behavior or setup changes
- Include testing notes for UI or runtime changes

## Pull Request Notes

A good PR should explain:

- what changed
- why it changed
- how it was tested
